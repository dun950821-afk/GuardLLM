# 国舜大模型安全护栏检测平台登录页

这是一个静态 HTML 登录页还原稿，只包含页面结构和视觉效果，不包含登录功能。

## 文件说明

- `index.html`：登录页面，可直接用浏览器打开。
- `assets/logo.png`：国舜 logo。
- `assets/hero-illustration.png`：左侧 3D 品牌视觉素材。
- `assets/reference-design.png`：生成稿参考图，用于对比还原效果。

## 使用方式

直接打开 `index.html` 即可预览。

如集成到项目，可将 HTML 中的 CSS 拆分到样式文件，并将 `assets` 目录复制到静态资源目录。
